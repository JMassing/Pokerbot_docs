static

filters contours after given method. Value ist given according to method.
Methods are:

  • LARGEST_AREA = 0, only keept largest contour
  • SMALLEST_AREA = 1, only keep smalles contour
  • LE_AREA = 2, contour area is smaller/equal then given value
  • GE_AREA = 3, contour area is larger/equal then given value
    Parameters
    contourContours found in Image.
    methodFiltering method.
    valueOnly for method 2 and 3. Gives max/min contour area.
